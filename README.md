A lightweight front-end repo for a **Library Management System** designed with a **Pinterest-inspired masonry layout**.

## Features
- Pinterest-like card grid for books.
- Category filter chips (Fiction, Science, History, Technology, Self-help).
- Live search by title, author, or genre.
- Availability status badges and quick book metadata.
- Responsive layout for desktop and mobile.

## Run locally
Because this project is static HTML/CSS/JS, you can open it directly or serve it:

```bash
python3 -m http.server 4173
```

Then visit:

- `http://localhost:4173`

## Stack
- HTML
- CSS (masonry columns)
- Vanilla JavaScript
